# FlowNeuro

FlowNeuro is an open-source development organization maintaining the Flow media player and its underlying recommendation engine. The project focuses on providing personalized content discovery entirely on-device, without relying on centralized servers or data tracking.

## Projects

* **[Flow (Android)](https://github.com/FlowNeuro/Flow):** An Android client built with Kotlin and Jetpack Compose. It features the `FlowNeuroEngine` for local recommendation generation, background playback, and a distinct YouTube Music interface.
* **[Flow](https://github.com/FlowNeuro/Flow-desktop):** A desktop client built with Rust and Tauri.
